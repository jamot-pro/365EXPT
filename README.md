# 365EXPT

> Can Artificial Intelligence create more meaningful work than it replaces?

We don't know the answer. **That's why this repository exists.**

365EXPT is a global, open-source experiment — a public laboratory where
we find out together. Every discussion, every prototype, every
experiment, and every discovery lives here, in the open.

---

## The experiment in 60 seconds

Most conversations about AI and work are about **replacement**. This
experiment tests the opposite bet:

> **AI's greatest contribution is not replacing humans. It is
> orchestrating human potential.**

So we run real experiments: real problems from real people, matched
with contributors who want to solve them, using AI as an assistant —
never as the decision-maker. Whatever we learn — successes *and*
failures — gets published here for everyone.

- **Everything is public.**
- **Everything is experimental.**
- **Everything can evolve.**
- **People come before technology.**
- **AI assists. Humans decide.**
